# MiniProjetJava-GestionSmartBudgetPersonnel
Projet de rattrapage en Programmation Java consistant à développer une application console de gestion de budget personnel. L'application permet à un utilisateur d'enregistrer ses revenus et ses dépenses, d'analyser sa situation financière, de calculer son épargne potentielle et d'obtenir des recommandations pour une meilleure gestion de son budget.
